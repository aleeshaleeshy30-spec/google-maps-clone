# Google Maps Clone

A modern, responsive Google Maps web clone built for **CWI TASK : 75**.

## ✨ Features

- Interactive Leaflet map
- Location search
- Directions and route simulation
- Markers and saved locations
- User authentication
- Responsive design

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Leaflet.js
